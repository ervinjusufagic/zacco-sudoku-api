## Sudoku-api
A node.js app built with express, moongose and mongoDB. Sends puzzles [zacco-sudoku](https://github.com/ervinjusufagic/zacco-sudoku)

### `Run the client`

1. clone project
2. npm install in root directory
3. node server.js
